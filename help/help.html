The File Menu

Open

Display the open file dialog to select a file to open. File is parsed and displayed in the application

New

Display the save file dialog so you can give a name for the new file. The new file is then created containing a single list with a single item

Save

Saves the current file

Save As...

Display the save file dialog so you can give a name for the file to save as

Exit

Exits the application. Will detect a dirty file and ask to save

Headings Popup Menu

Add a new heading to the headings list (keyboard shortcut is h)

Edit Heading

Edit the heading text

Remove Heading

Checklist Popup Menu

Add Item

Add a new item to the check list (keyboard shortcut is i)

Edit Item

Edit the check list item text (keyboard shortcut is e)

Remove Item

Remove the item from the list. If you remove the last item in the check list it will remove the heading from the headings list

Reverse List

This action will reverse the order of the items in the check list

Renumber List

This action will renumber the check list items

Check List Format

* One or more lists can be included in a file
* Title must be the first line of the file
* Headings are in uppercase and unnumbered
* List items should start with a number
** List items are automatically numbered when saved
* Sublist items must start with ... (3 dots) to prevent numbering on save
* Comment lines myst start with # (hash symbol) to prevent numbering on save
