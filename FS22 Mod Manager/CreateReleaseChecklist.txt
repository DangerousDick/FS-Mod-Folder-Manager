Before creating release and generating the MSI file...
* Set applicaiton version in MainForm.cs file
* Update github wiki to reflect changes if required (https://github.com/DangerousDick/FsModsManager/wiki)
* Update Setup project version number (otherwise old version will not be uninstalled)
* Tag and push all changes to github
* Generate MSI file
* Copy MSI file to FsModManager project, tag and push to git hub
* Create releases for both fs22-mods-manager and FsModManager projects in github
