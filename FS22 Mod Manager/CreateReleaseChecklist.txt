Before creating release and generating the MSI file...
* Set application version in MainForm.cs file
* Update github wiki to reflect changes if required (https://github.com/DangerousDick/FS-Mod-Folder-Manager/wiki)
* Using the Advanced installer application
    * Update version in the project settings
    * Generate MSI file
    * MSI file should be copied to solution root folder automatically
* Tag and push all changes to github
* Create releases for both fs22-mods-manager and FsModManager projects in github
